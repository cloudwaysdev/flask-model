- Install pipenv
`pip install --user pipenv`

- Install from Pipfile, if there is one:

`pipenv install`

- Activate the Pipenv shell

`pipenv shell`

- Create database in mysql

- Populate database

`flask db upgrade`

- Start flask application

`./run.sh`
